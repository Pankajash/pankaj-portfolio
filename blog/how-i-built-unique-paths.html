How I Built the Unique Paths Calculator

2024-10-05

I implemented a dynamic programming calculator that computes the number of unique paths from the top-left to the bottom-right of a grid. The project was built with HTML, CSS, and JavaScript, and uses a clean DP table approach to visualize algorithmic progression.

Later, I enhanced it with AI-generated explanations using the Gemini API, making it more educational and interactive for learners exploring dynamic programming.
